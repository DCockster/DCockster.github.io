Notes is very simple note pad used to jot down notes on your android devise. Built on a PRIVATE MODE so files only to be accessed from within the app. Text and links are selectable for your convenience, plus links are highlighted blue. This app only creates a text file currently.This is one of the first apps I built and honestly I built it on my phone one day because I was bored.I later added the project to android studio to improve the build.I might revisit the app one day to add more things but lets be honest google play store has many options when it comes to note pads.

System requirements:

  • Android 5.0 +

Permissions required:

  • Storage Permission will be required to access your files located on your devise.
